# KREM Editor

KREM Editor is a lightweight experimental code editor built for learning, testing UI systems, and developing custom tooling.

Currently in **Alpha Dev Stage** — unstable and under active development.

---

## 🚧 Status
Alpha Build (v0.1-dev)

Expect bugs, unfinished features, and breaking changes.

---

## 🎯 Purpose
This project is focused on:
- Building a custom code editor architecture
- Experimenting with UI/UX systems
- Testing performance and rendering approaches
- Developing modular tooling systems for future expansion

---

## ⚠️ Warning
This is an early-stage experimental project.

It may:
- crash unexpectedly  
- behave inconsistently  
- contain incomplete features  
- change structure without notice  

Do not use it in production environments.

---

## 🧪 Features (planned / experimental)
- Basic text editor core
- Custom UI system
- File handling (WIP)
- Syntax highlighting (planned)
- Plugin system (future)

---

## 📦 Build Status
No stable releases yet.

Source-only development version.

---

## 🧠 Notes
KREM Editor is not meant to compete with professional IDEs at this stage.
It is a long-term experimental project evolving over time.
